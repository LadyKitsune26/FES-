    2.3 Common Properties

    Most Common Properties

  • color - color of text
  • background - Background of element
  • width - width of elements
  • height - height of elements
  • border - border around element
  • font - size, weight, family (i.e: bold, normal, italics)

    • 3 ways to set color in CSS.

  • Words: red, green, blue, cyan.
  • RGBC: rgb(o, 255, 0) - green
  • Hex: #ff0000 - Red.

    • 2.4 Specificity

    Specificity - the priorization of the styling of the elements. Order of operations.

  • Class is prioritized over elements.
  • ID's are higher then class, only one ID per document.
  • # to target ID's, . to target classes.
  • Elements>Class>ID's
