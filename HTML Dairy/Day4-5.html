2.6 Complex Properties in css

Display, Inline, Position

Today(yesterday) I learn about complex CSS. Which is really about placing things on the page in the way you want.

Display

Display - a block that takes up 100% of the width and starts on a new line.

Inline

Doesn't allow you to set a width or hieght for an element. Starts on the same line as an element. eg. span or img tags.

Position

CSS Position Values (common). static, followed the normal flow of the page. Relative follows the normal flow of the page but also lets you position using top, left, right, and bottom. Absolute: Does not follow normal flow of the page, Lets you position with top, left, bottom and right, this is based off the closest position relative to the parent.

Uncommon CSS position values

Fixed - sticks elements to specific position on the page. Can use top, left, bottom and right. Sticky - sticks elements to a page perminently, once it has a threshold.

2.7 Media Queries

Media Queries are used for if you want to also use a tablet and how the webpage responds to alternative veiwing options.

  • 576px - Mobile Phones
  • 768px - Tablets
  • 992px - Destops
  • 1200px - Large Destops
